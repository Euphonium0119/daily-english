# Changelog

## v1.3.0 (2026-07-23)

### 🎨 邮件视觉设计全面升级

- **每日主题色轮换**：7 套调色板按星期自动切换（周一深海蓝 / 周二森林绿 / 周三暖陶土 / 周四暮光紫 / 周五石墨蓝 / 周六玫瑰金 / 周日琥珀金），每套含 Header 渐变、背景、强调色、卡片色完整配色
- **字体分层**：UI 元数据（标签、页脚、卡片字段）改用系统无衬线字体栈，正文阅读区保留 Playfair Display + Georgia 衬线组合
- **视觉层次优化**：三主区块编号 ①②③、分隔线改为 dotted 虚线、Quote 区块加专属背景色
- **移动端响应式**：外层 padding 24→10px，内容块内边距同步缩小，正文可用宽度增加约 30%
- **词汇卡片信息层级简化**：三级递进（单词释义 → 定义例句 → 📋 更多收纳搭配/派生/同根/近义）
- **首字母下沉**：原文第一段首字 drop cap + 顶部 3px 主题色强调线

### 🤖 AI 分析与内容质量增强

- **Prompt 重构**：结构化 ROLE / TASK / OUTPUT FORMAT / CONSTRAINTS 分块，降低歧义
- **新增输出字段**：`variants`(词形变化)、`category`(文章主题分类)、`grammar_tip`(语法小贴士)、`word_of_the_day`(每日一词)
- **新增邮件区块**：📐 语法小贴士（语法点 + 文中原句 + 中文解释）、🌟 每日一词（精选单词 + 学习理由）
- **词形变化高亮**：利用派生词自动扩展正则匹配，`measure → measures / measured / measuring` 全部高亮
- **文章筛选复合评分**：5 维评分（长度接近度 40% + 有作者 10% + 优质新闻源 20% + 标题质量 10% + 时效性 20%）替代纯长度排序
- **并行文章抓取**：`ThreadPoolExecutor` 5 线程并发，抓取速度提升约 3 倍
- **API 重试机制**：NewsAPI / DeepSeek 各 3 次退避重试
- **NEWS_SOURCES 环境变量化**：支持自定义新闻源

### ⚙️ 参数调整

- DeepSeek temperature: `0.3` → `0.1`（提升 JSON 输出稳定性）
- 文章质量预过滤：跳过 `[Removed]` 标题和无标题文章

## v1.0 (2026-07)

- 初始发布：每日英文新闻抓取 → DeepSeek AI 翻译分析 → HTML 邮件推送
- 核心功能：双语逐段精读、重点词汇卡片（音标/释义/例句/搭配/派生/同根/近义）、长难句拆解、每日金句
- 新闻源：BBC News, CNN, The Guardian, Reuters, Associated Press, ABC News
- 定时推送：GitHub Actions 北京时间每天 7:00 自动发送
